# Python Standards

Python is the mandatory default language for ALL SPECTRA development unless technically impossible.

## Language Choice

- **Default**: Python/PySpark
- **Business logic**: Python
- **CLI tools**: Python
- **APIs**: Python
- **Automation**: Python (not bash)
- **Infrastructure scripts**: Python (not PowerShell)
- **Testing**: pytest

Bash is only acceptable for trivial wrappers (<20 lines).
