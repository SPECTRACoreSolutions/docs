# SPECTRA Portal

Public-facing website and portal for SPECTRA.

## Overview

The portal is built with:

- **Astro** - Modern web framework
- **Tailwind CSS** - Styling
- **Auth.js** - Authentication

## Features

- OAuth authentication (Google, Apple)
- Protected dashboard
- Responsive design
- Real-time chat integration
