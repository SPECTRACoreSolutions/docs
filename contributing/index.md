# Contributing to SPECTRA

Thank you for contributing to SPECTRA!

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Write tests (if applicable)
5. Submit a pull request

## Documentation

- [Documentation Guide](documentation.md) - How to contribute documentation
- [Code Style](code-style.md) - Coding standards and style guide

## Standards

- Follow Python standards (see [Standards](../standards/index.md))
- Write tests for new features
- Update documentation as needed
- Follow naming conventions
