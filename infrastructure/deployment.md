# Deployment Guides

Deployment guides for SPECTRA services and applications.

## Railway

Railway is the recommended platform for quick cloud deployments.

### Quick Deploy

1. Connect repository to Railway
2. Configure environment variables
3. Deploy!

See [Railway Deployment Guide](../../Core/labs/tooling/docs/railway-deployment.md) for details.

## GitHub Pages

Documentation and static sites deploy automatically to GitHub Pages.

## Docker

All services include Dockerfiles for containerized deployment.
